# KS_Troop_Calculator
Calculator to help make easy troop numbers to match your total troops, total marches and commonly used troop ratios.

## Features
- Calculates troops per march
- Supports Infantry, Cavalry, and Archers
- Ratio-based distribution
- Designed for mobile use
- Shareable via Discord

## Usage
Open the calculator link, enter your total troops, number of marches, and desired ratios.
